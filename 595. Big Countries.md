# Problem
https://leetcode.com/problems/big-countries/

# Intuition
- it says in the question you need an OR condition

# Solution 1

```sql
select 
  name, population, area
from 
  world
where 
  population >= 25000000

union

select 
  name, population, area
from 
  world
where 
  area >= 3000000
;
```

# Solution 2 

```sql
select 
  name, 
  population, 
  area
from 
  world
where 
  population >= 25000000
  OR area >= 3000000
;
```
